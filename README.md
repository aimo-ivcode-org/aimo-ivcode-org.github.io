# Aimo Pages

This is the source for the project site for Aimo. It's built with Jekyll and Bulma Clean Theme.

 - [Jekyll](https://jekyllrb.com/)
 - [Bulma Clean Theme](https://github.com/chrisrhymes/bulma-clean-theme)

# Setup
Prerequisites:
 - Ruby `~> 3.2`
 - Bundler `~> 2.7`

After installing Ruby, run the following command to install bundler and jekyll:
```bash
gem install bundler jekyll
```

# Run
To run the project locally, navigate to the project directory and run the following command:

```bash
bundle install
bundle exec jekyll serve
```

Once the server is running, you can access the site at http://localhost:4000
